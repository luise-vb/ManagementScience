Project 05 / Linear Programming

Street Food Truck

Maximising revenue across a mixed fleet of taco and burrito trucks under budget and availability constraints

01 — Overview

The Fleet Mix Problem

A street food company wants to maximise its daily revenue by deciding how many taco trucks and burrito trucks to operate. Each truck type generates different revenue but costs different amounts and has limited availability. This is a classic linear programming problem — find the optimal mix under constraints.

This problem is a second application of LP, reinforcing the method from the Pinocchio Problem (Project 1) but in a different operational context: fleet management vs. production planning. The same graphical technique — feasible region, corner points, iso-profit line slope — applies.

03 — Evidence from My Work

Problem Formulation & Solution

Decision variables:

Variables x₁ = number of taco trucks  ·  x₂ = number of burrito trucks
Objective Function — Maximise Z = 2,000 x₁ + 5,000 x₂

Constraints from class notes:

Budget Constraint 30,000 x₁ + 40,000 x₂ ≤ 360,000  ·  Simplified: 3x₁ + 4x₂ ≤ 36
Availability Constraints x₁ ≤ 8 (max 8 taco trucks)  ·  x₂ ≤ 6 (max 6 burrito trucks)  ·  x₁, x₂ ≥ 0

Boundary lines for graphical solution:

  • ·3x₁ + 4x₂ = 36

04 — What Did I Learn?

Knowledge, Skills & Reflection

Knowledge

Repeating LP in a new context (fleet vs production) cemented the method. The slope of the objective function tells us the rate at which one variable can substitute for another while keeping profit constant — a powerful sensitivity concept.

Skills

I practised simplifying constraints (dividing through by 10,000), plotting boundary lines, identifying corner points algebraically, and computing the objective function slope to match the iso-profit line to the feasible region.

Reflection

It was useful to see LP applied to a fleet/service context rather than manufacturing. The model clearly shows that burrito trucks generate more revenue per unit — so the optimal solution maxes them out first, then fills remaining budget with taco trucks.
